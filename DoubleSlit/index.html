Double Slit Experiment Simulator

Inspired by 3Blue1Brown-style visuals: tune slit geometry and wavelength to watch elegant wave interference emerge from simple math.

Model (Fraunhofer Approximation)

We model the scalar wave on a screen at distance L using small angles: sin θ ≈ x / L, sin φ ≈ y / L.

ψ(x, y, t) = A · sinc(βx) · sinc(βy) · 2 cos(α) · exp(i(kL - ωt))

βx = π a x / (λ L)      βy = π h y / (λ L)      α = π d x / (λ L)
k = 2π/λ

Intensity: I(x, y) = |ψ|² = 4 A² sinc²(βx) sinc²(βy) cos²(α)

a = slit width, h = slit height, d = slit separation, λ = wavelength.
